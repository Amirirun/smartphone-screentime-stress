# smartphone-screentime-productivity


Beschreibung des Datensatzes

Für dieses Projekt wird der Datensatz „Smartphone Usage, Productivity & Lifestyle Analytics Dataset“ verwendet. Der Datensatz enthält 50’000 anonymisierte Einträge zur Smartphone-Nutzung und deren Zusammenhang mit Produktivität, Schlafverhalten und Stresslevel. Enthalten sind unter anderem die Variablen Alter, Geschlecht, Beruf, Gerätetyp, tägliche Smartphone-Nutzungsdauer, Social-Media-Nutzung, Schlafstunden, Stresslevel, Koffeinkonsum sowie Wochenend-Bildschirmzeit.

Die Zielvariable dieses Projekts ist der Work_Productivity_Score, welcher mittels Regressionsanalyse vorhergesagt wird.

Datenschutz

Der Datensatz enthält keine personenbezogenen Daten. Es sind weder Namen, Adressen noch Kontaktinformationen vorhanden.
